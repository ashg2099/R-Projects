Property Transactions (Victoria 2021):
1. Cleaned and preprocessed property transaction data using shell scripting to analyze price trends, suburb frequencies, and property types.
2. Filtered data for specific conditions like area size, property type, and transaction months, while conducting textual analysis on property features.

Olympic Tweets Analysis:
1. Scraped data from Twitter to analyze over 100,000 tweets related to the Tokyo 2020 Olympics.
2. Processed and cleaned text data using tidytext, hunspell, and dplyr packages for sentiment and keyword analysis.
3. Analyzed geographic engagement, identifying top tweeting countries and user behavior trends based on tweet length and content.
4. Visualized key insights using ggplot2, including trends in tweet activity over time, the frequency of user mentions, and the most common terms used in discussions about the Olympics.
